#  Sales Dashboard | Power BI Project

## Project Overview

This project focuses on building an interactive Sales Dashboard using Power BI to analyze business performance and generate actionable insights. The dashboard helps stakeholders monitor KPIs, identify trends, and make data-driven decisions.

---

##  Problem Statement
Businesses often struggle to track sales performance across regions and products due to scattered data.
This dashboard helps in centralized monitoring and analysis.

---

##  Solution
Developed an interactive Power BI dashboard to centralize and visualize key metrics.

---

##  Business Objective
* Track overall sales performance
* Identify top-performing regions and products
* Analyze monthly and yearly sales trends
* Improve decision-making using visual insights

---

##  Key Features

* Dynamic KPI indicators (Total Sales, Profit, Quantity)
* Region-wise and category-wise performance analysis
* Monthly sales trend visualization
* Top-performing products identification
* Interactive filters and slicers for detailed analysis

---

##  Tools & Technologies

* Power BI (Data Visualization)
* DAX (Data Analysis Expressions)
* Excel / CSV (Data Source)

---

##  Key Insights

* Identified top revenue-generating regions contributing maximum sales
* Observed seasonal trends in monthly sales performance
* Highlighted underperforming categories needing improvement
* Derived insights for better inventory and sales strategy planning
